Beetl Eclipse 插件安装方式

  • 启动Eclipse

  • 打开菜单栏按以下菜单路径依次打开

    Help -> Install New Softwave… ->点击Add按钮弹出一个对话框

  • 弹出的对话框中Name随意填写，如填写“beetl”，Location请填写

    http://ibeetl.com/eclipse/

  • 选中您要安装的Beetl Eclipse Plugin，按提示依次Next，直至Finish重启Eclipse即可

Beetl Eclipse 插件卸载方式

  • 启动Eclipse

  • 打开菜单栏按以下菜单路径依次打开

    Help -> Install New Softwave… ->找到 already installed 超链接打开一个对话框

  • 搜索“beetl”，找到beetl插件选中，点击底部的Uninstall...按钮按操作进行卸载即可
