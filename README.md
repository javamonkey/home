### Beetl官方网站：[http://ibeetl.com](http://ibeetl.com)

Beetl是一款秒杀FreeMarker的高性能Java模板引擎，中文文档，简单容易上手。

BeetlSQL是一款基于Beetl的，高性能的全功能的Java Dao工具，同时具有Hibernate和MyBatis的优点，适用于承认以 SQL 为中心，同时又需求工具能自动能生成大量常用的 SQL 的应用。
